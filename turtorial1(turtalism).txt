Turtelism simluator

- install turtalism
sudo apt update 
sudo apt install ros-humble-turtlesim

- verify instalition
ros2 pkg executables turtlesim


- start turtalism
ros2 run turtlesim turtlesim_node

- open a new terminal
ros2 run turtlesim turtle_teleop_key

- now you can control the turtle with arrows
